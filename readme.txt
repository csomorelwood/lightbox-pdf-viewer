With this plugin you will be able to view PDF files in lighbox popup, and play mp3 files via shortcodes.

== Description ==

Usage:

You just simply need to put the shortcode into your site, and it will do the job.

1. Upload your PDF to Media Library
2. Make a new post at the Lightbox PDF menu
3. Set the title, thumbnail, select an mp3 file and select a pdf file
4. Check the ID of the created PDF post
5. Put the shortcode wherever you need

The shortcode looks like this: 
[lbpdfviewr id={ID of the LightBox PDF you want} type={button / thumbnail} audio={true / false} class="{custom css class}"]
